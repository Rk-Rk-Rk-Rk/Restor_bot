🍽 Ресторан бот 
Телеграм-бот для бронирования столиков в ресторане.

✨ Возможности для гостей

Интерактивное меню: Просмотр списка свободных столиков с указанием мест.

Моментальная бронь: Бронирование в один клик.

Совместный заказ для компаний

🛠 Для администратора

Управление всеми фунциями бота

Статусы: Ручное изменение статуса, сброс всех броней.

База данных: Все настройки сохраняются в SQLite (файл restaurant.db).


 Структура проекта
 
bot.py — Основная логика бота.

database.py — Работа с SQLite.

config.py — Конфигурация.

restaurant.db — Файл базы данных.
